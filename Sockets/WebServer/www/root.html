You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply?num1=1&num2=2 -- returns the product of two integers (invalid parameters use default values)
  • /convertTo?decimal=1&format=x -- converts decimal to binary (b), hexadecimal (h)
